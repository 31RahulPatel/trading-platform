# Services Directory

This directory contains all microservices for the trading platform:

- `auth-service/` - Authentication and authorization service
- `order-service/` - Order management and processing
- `portfolio-service/` - Portfolio tracking and analytics  
- `admin-service/` - Administrative dashboard and user management
- `order-worker/` - Background job processing for orders

Each service will have its own:
- `src/` - Source code
- `tests/` - Test files
- `Dockerfile` - Container configuration
- `package.json` - Dependencies and scripts
- `README.md` - Service-specific documentation
